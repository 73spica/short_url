# Short URL Works
短縮URL関連のリポジトリ

# Environment
- Python3.6.3
- pip (9.0.1)
- lxml (4.1.1)
- requests (2.18.4)
- beautifulsoup4 (4.6.0)

# Setup
```
$ python -m venv work-env
$ source ./work-env/bin/activate
$ pip install requests
$ pip install beautifulsoup4
$ pip install lxml

$ deactivate
```

# Usage
```
$ source ./work-env/bin/activate
$ python ~~~.py
```
